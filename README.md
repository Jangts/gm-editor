# GM Editor

一个非常简单的markdown编辑器。
用于学习[Atom Shell](https://github.com/atom/atom-shell)

### How To Build
```bash

npm install gulp -g
npm install

gulp

# OS X:
/path/to/Atom.app/Contents/MacOS/Atom /path/to/gm-editor
```
